# UNIRAW • Media Pool (Original UI)

This project restores your original React UI with **the same tile scale, preview height,
sidebar width, colors, and icons**. It uses Vite + React + Tailwind, with no visual changes
beyond the minimal scaffolding to run it.

## Run
```bash
npm install
npm run dev
# open the URL shown (usually http://localhost:5173)
```
